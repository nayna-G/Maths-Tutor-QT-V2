# 🧮 മാത്സ്-ട്യൂട്ടർ ഉപയോക്തൃ ഗൈഡ് (User Guide)

മാത്സ്-ട്യൂട്ടർ (Maths-Tutor) എന്നത് അന്ധരും കാഴ്ചപരിമിതിയുള്ളവരുമായ കുട്ടികൾക്കായി രൂപകൽപ്പന ചെയ്ത ഒരു സൗജന്യ ഓപ്പൺ സോഴ്സ് ഗണിത പഠന ആപ്ലിക്കേഷനാണ്. കീബോർഡും ശബ്ദവും മാത്രം ഉപയോഗിച്ച് പ്രവർത്തിപ്പിക്കാൻ കഴിയുന്ന രീതിയിലാണ് ഇത് നിർമ്മിച്ചിരിക്കുന്നത്. അതുകൊണ്ട് തന്നെ ഇതിന് മൗസിന്റെയോ സ്ക്രീനിന്റെയോ ആവശ്യമില്ല. ഓരോ ചോദ്യവും, ഉത്തരവും, മെനു ഓപ്ഷനുകളും ബിൽറ്റ്-ഇൻ ടെക്സ്റ്റ്-ടു-സ്പീച്ച് (TTS) എൻജിൻ വഴി ഉച്ചത്തിൽ വായിച്ചു കേൾപ്പിക്കുന്നു, ഇത് ഓരോ കുട്ടിക്കും സ്വതന്ത്രമായി പഠിക്കാൻ അവസരമൊരുക്കുന്നു.

## ⚡ പെട്ടെന്ന് തുടങ്ങാൻ

1. നിങ്ങളുടെ കമ്പ്യൂട്ടറിൽ മാത്സ്-ട്യൂട്ടർ ആപ്ലിക്കേഷൻ തുറക്കുക.
2. ഭാഷ തിരഞ്ഞെടുക്കുന്നതിനുള്ള അറിയിപ്പ് ശ്രവിക്കുക, നിങ്ങളുടെ പ്രിയപ്പെട്ട ഭാഷ തിരഞ്ഞെടുത്ത ശേഷം **Enter** അമർത്തുക.
3. മൂന്ന് ഗെയിം മോഡുകൾക്കിടയിൽ മാറാൻ **Tab** കീ ഉപയോഗിക്കുക: **Quickplay (ക്വിക്ക്പ്ലേ)**, **Game Mode (ഗെയിം മോഡ്)**, **Learning Mode (പഠന മോഡ്)**. തിരഞ്ഞെടുക്കാൻ **Space Bar** അല്ലെങ്കിൽ **Enter** അമർത്തുക.
4. ആപ്പ് നിങ്ങൾക്ക് ആദ്യത്തെ ചോദ്യം വായിച്ചു തരുന്നത് ശ്രദ്ധിച്ചു കേൾക്കുക. നമ്പർ കീകൾ ഉപയോഗിച്ച് നിങ്ങളുടെ ഉത്തരം ടൈപ്പ് ചെയ്യുക.
5. ഉത്തരം സമർപ്പിക്കാൻ **Enter** കീ അമർത്തുക. *(ശ്രദ്ധിക്കുക: Enter കീ മാത്രമേ ഉത്തരം സമർപ്പിക്കാൻ സഹായിക്കൂ; Space Bar ഇതിനായി ഉപയോഗിക്കാൻ കഴിയില്ല)*. ചോദ്യം വീണ്ടും കേൾക്കണമെന്നുണ്ടെങ്കിൽ, ഒരേ സമയം **Control + R** കീകൾ അമർത്തുക.

## 🛠 ഇൻസ്റ്റാളേഷനും സജ്ജീകരണവും

`[രക്ഷിതാക്കൾക്കും അധ്യാപകർക്കും]`

മാത്സ്-ട്യൂട്ടർ സജ്ജീകരിക്കുന്നതിന് താഴെ പറയുന്ന ലളിതമായ ഘട്ടങ്ങൾ പാലിക്കുക:

1. നിങ്ങളുടെ ടെർമിനലിൽ ഈ കമാൻഡ് പ്രവർത്തിപ്പിക്കുക:
   ```bash
   pip install -r requirements.txt
   ```
2. നിങ്ങളുടെ സിസ്റ്റം സോഫ്റ്റ്‌വെയർ മാനേജർ വഴി `espeak-ng` ഇൻസ്റ്റാൾ ചെയ്യുക (ഓഫ്‌ലൈൻ സ്പീച്ചിനായി).
3. ആപ്ലിക്കേഷൻ റൺ ചെയ്യുക:
   ```bash
   python main.py
   ```

## 🚀 ആദ്യമായി തുറക്കുമ്പോൾ

നിങ്ങൾ ആദ്യമായി ആപ്ലിക്കേഷൻ തുറക്കുമ്പോൾ, ഒരു ഭാഷാ തിരഞ്ഞെടുപ്പ് വിൻഡോ ദൃശ്യമാകും.
* ഭാഷകളുടെ പട്ടികയിലൂടെ നീങ്ങാൻ **ആരോ കീകൾ (Arrow Keys)** ഉപയോഗിക്കുക. വോയ്‌സ് അസിസ്റ്റന്റ് ഇവ അനൗൺസ് ചെയ്യും: **English**, **Hindi (हिंदी)**, **Malayalam (മലയാളം)**, **Tamil (தமிழ்)**, **Arabic (عربي)**, **Sanskrit (संस्कृत)**.
* "Remember my selection" (എന്റെ ഭാഷ ഓർത്തു വെക്കുക) എന്ന ചെക്ക്ബോക്സിലെത്താൻ ഒരു തവണ **Tab** കീ അമർത്തുക. ഇത് ചെക്ക് ചെയ്യാനോ അൺചെക്ക് ചെയ്യാനോ **Space Bar** അമർത്തുക.
* തുടർന്ന് **Continue** ബട്ടണിലെത്താൻ വീണ്ടും **Tab** അമർത്തുക, ശേഷം **Space Bar** അല്ലെങ്കിൽ **Enter** അമർത്തുക. *(നിങ്ങൾക്ക് ഏത് സമയത്തും **Enter** കീ അമർത്തിക്കൊണ്ട് നേരിട്ട് ഭാഷ ഉറപ്പിക്കാവുന്നതാണ്)*.
നിങ്ങളുടെ പ്രിയപ്പെട്ട ഭാഷ ഉറപ്പിച്ച ശേഷം, സ്വാഗതം ചെയ്തുകൊണ്ട് "Welcome to Maths Tutor!" എന്ന വോയ്‌സ് അറിയിപ്പ് കേൾക്കാം.

### പ്രധാന മെനുവും ഓഡിയോ നിയന്ത്രണങ്ങളും

പ്രധാന സ്ക്രീനിൽ **Tab** കീ ഉപയോഗിച്ച് ഓരോ ബട്ടണുകളിലൂടെയും നീങ്ങുമ്പോൾ ശബ്ദങ്ങൾ കേൾക്കാവുന്നതാണ്.

**ഓഡിയോ മ്യൂട്ട് (Audio Mute)**
**Tab** അമർത്തി നിങ്ങൾക്ക് ഓഡിയോ ടോഗിളിൽ എത്താം. പശ്ചാത്തല സംഗെതവും ശബ്ദങ്ങളും പ്രവർത്തിക്കുന്നുണ്ടെങ്കിൽ സ്ക്രീൻ റീഡർ "Audio Unmuted" എന്നും, ഇല്ലെങ്കിൽ "Audio Muted" എന്നും അനൗൺസ് ചെയ്യും. ഇവ തമ്മിൽ മാറാൻ **Space Bar** അല്ലെങ്കിൽ **Enter** അമർത്തുക.

**തീം ടോഗിൾ (Theme Toggle)**
ആപ്ലിക്കേഷന്റെ മുകളിൽ ഇടതുഭാഗത്തായി ലൈറ്റ് (Light - 🌙), ഡാർക്ക് (Dark - ☀️) തീമുകൾ തമ്മിൽ മാറാനുള്ള ഓപ്ഷനുണ്ട്. ഇത് ഭാഗികമായി കാഴ്ചയുള്ളവർക്ക് കൂടുതൽ കോൺട്രാസ്റ്റ് നൽകാൻ സഹായിക്കുന്നു.

## 🎮 പഠന രീതി (മോഡ്) തിരഞ്ഞെടുക്കൽ

പ്രധാന മെനുവിലൂടെ നീങ്ങാൻ **Tab** കീ അമർത്തുക. ഓരോ തവണയും സ്ക്രീൻ റീഡർ ലഭ്യമായ ബട്ടണുകളുടെ പേര് അനൗൺസ് ചെയ്യും. നിങ്ങൾക്ക് ആവശ്യമുള്ളത് തിരഞ്ഞെടുക്കാൻ **Space Bar** അല്ലെങ്കിൽ **Enter** അമർത്തുക.
*(മുകളിൽ നിന്നുള്ള ഓർഡർ: തീം ടോഗിൾ, ക്വിക്ക്പ്ലേ, ഗെയിം മോഡ്, പഠന മോഡ്, ഓഡിയോ ടോഗിൾ, അപ്‌ലോഡ്, ക്രമീകരണങ്ങൾ).*

### ⚡ ക്വിക്ക്പ്ലേ (Quickplay)
ഈ മോഡിൽ നിങ്ങൾക്ക് തുടർച്ചയായി കുറച്ച് ഗണിത ചോദ്യങ്ങൾ കേൾക്കാം. പശ്ചാത്തലത്തിൽ ഒരു ടൈമർ പ്രവർത്തിക്കുന്നുണ്ടാകും, കൂടാതെ എത്ര വേഗത്തിൽ നിങ്ങൾ ഉത്തരം നൽകുന്നു എന്നതിനനുസരിച്ച് ശബ്ദ ഫീഡ്‌ബാക്ക് ലഭിക്കും.

### 🎓 പഠന മോഡ് (Learning Mode)
ഈ മോഡിൽ സമയപരിധിയില്ല. ഇത് കുട്ടികളുടെ ആത്മവിശ്വാസം വർദ്ധിപ്പിക്കാൻ സഹായിക്കുന്നു. ഇത് തിരഞ്ഞെടുത്ത ശേഷം വിഷയങ്ങൾ കേൾക്കാൻ **Tab** അമർത്തുക, തുടർന്ന് അനുയോജ്യമായ വിഷയം തിരഞ്ഞെടുക്കാൻ **Enter** അമർത്തുക:
* **കഥാ രൂപത്തിലുള്ളവ (Story)**: വാക്യങ്ങളായുള്ള ഗണിത ചോദ്യങ്ങൾ ഇവിടെ കേൾക്കാം. ഉദാഹരണത്തിന്: "രാജു തന്റെ കൂട്ടുകാർക്ക് പങ്കിടാൻ 27 മിട്ടായികൾ കൊണ്ടുവന്നു. അവൻ അതിൽ 10 എണ്ണം കൊടുത്തു. ഇപ്പോൾ രാജുവിന്റെ പക്കൽ എത്ര മിട്ടായികളുണ്ട്?"
* **സമയം (Time)**: ക്ലോക്ക് റീഡിംഗ്, സമയം കണക്കാക്കൽ എന്നിവയുമായി ബന്ധപ്പെട്ട ചോദ്യങ്ങൾ.
* **കറൻസി (Currency)**: പണം, ഷോപ്പിംഗ്, ബാക്കി തുക എന്നിവയുമായി ബന്ധപ്പെട്ട വെല്ലുവിളികൾ.
* **ദൂരം (Distance)**: ദൂരവും അളവുകളും കണക്കാക്കുന്ന ചോദ്യങ്ങൾ.
* **ബെൽറിംഗ് (Bellring)**: ഒരു മണി ശബ്ദിക്കുന്നതിന്റെ എണ്ണം ശ്രദ്ധാപൂർവ്വം കേട്ട് അത് എത്ര തവണ ശബ്ദിച്ചു എന്ന് കണക്കാക്കുന്ന വിനോദം.
* **ഗണിത പ്രവർത്തനങ്ങൾ (Operations)**: സങ്കലനം (Addition), കുറയ്ക്കൽ (Subtraction), ഗുണനം (Multiplication), ഹരണം (Division), ശിഷ്ടം (Remainder), ശതമാനം (Percentage) എന്നിവയുടെ ലളിതമായ പരിശീലനം.

### 🎮 ഗെയിം മോഡ് (Game Mode)
കൂടുതൽ ആത്മവിശ്വാസമുള്ള കുട്ടികൾക്കായി രൂപകൽപ്പന ചെയ്ത ഈ മോഡ് ആരംഭിക്കുന്നത് ഒരു **കംപ്ലീറ്റ് വാംഅപ്പ് മാച്ചോടെ (Warmup Match)** ആണ്.
* ആപ്പ് വാംഅപ്പ് മാച്ചിനെക്കുറിച്ച് വിവരിക്കുകയും കുട്ടിയെ സഹായിക്കുകയും ചെയ്യും.
* എല്ലാ വിഭാഗങ്ങളിൽ നിന്നും കൂടി ആകെ 14 ചോദ്യങ്ങൾ നിങ്ങൾ കേൾക്കും.
* ആപ്പ് നിങ്ങളുടെ വേഗതയും കൃത്യതയും ട്രാക്ക് ചെയ്യുകയും, ഗെയിമിന്റെ പ്രയാസം നിങ്ങൾക്ക് അനുയോജ്യമായ രീതിയിൽ ക്രമീകരിക്കുകയും ചെയ്യും.
* ആരംഭിക്കാൻ **Begin Warmup** ബട്ടണിൽ **Space Bar** അല്ലെങ്കിൽ **Enter** അമർത്തുക.
പ്രധാന ഗെയിം മത്സരബുദ്ധിയോടെയുള്ളതാണ്. ഇതിൽ ലൈവ് സ്കോർ അനൗൺസ് ചെയ്യും. കൂടാതെ തെറ്റായ ഉത്തരങ്ങൾ നൽകിയാൽ ശരിയായ ഉത്തരം വെളിപ്പെടുത്തില്ല.

## 📊 കഠിന്യ നിലകൾ

ഇതിൽ അഞ്ച് തലത്തിലുള്ള വെല്ലുവിളികളുണ്ട്: **ലളിതം (Simple)**, **എളുപ്പം (Easy)**, **ഇടത്തരം (Medium)**, **കഠിനം (Hard)**, **വെല്ലുവിളി നിറഞ്ഞത് (Challenging)**. പ്രയാസം കൂടുന്തോറും അക്കങ്ങളുടെ എണ്ണം വർദ്ധിക്കും. നിങ്ങൾക്ക് ഏത് സമയത്തും **ക്രമീകരണങ്ങൾ (Settings)** മെനുവിൽ പോയി പ്രയാസ നില മാറ്റാവുന്നതാണ്.

## 📝 ചോദ്യങ്ങൾക്ക് ഉത്തരം നൽകൽ

1. ഒരു സെഷൻ ആരംഭിക്കുമ്പോൾ, ചോദ്യം സ്വയമേവ വായിച്ചു കേൾപ്പിക്കും.
2. ചോദ്യം ശരിയായി കേട്ടില്ലെങ്കിൽ വീണ്ടും കേൾക്കാൻ **Control + R** അമർത്തുക.
3. നമ്പർ കീകൾ ഉപയോഗിച്ച് നിങ്ങളുടെ ഉത്തരം ടൈപ്പ് ചെയ്യുക. വാക്യങ്ങളായുള്ള കഥാ ചോദ്യങ്ങൾ ശ്രദ്ധാപൂർവ്വം പൂർണ്ണമായി കേട്ട ശേഷം ഉത്തരം നൽകുക.
4. സമർപ്പിക്കാൻ **Enter** അമർത്തുക. *(ശ്രദ്ധിക്കുക: Enter കീ മാത്രമേ ഉപയോഗിക്കാവൂ, Space Bar ഉപയോഗിക്കരുത്)*.
5. ഉത്തരം തെറ്റായാലും ശരിയായാലും ആപ്പ് സ്വയമേവ അടുത്ത ചോദ്യത്തിലേക്ക് നീങ്ങും. അടുത്ത ചോദ്യത്തിനായി നിങ്ങൾ മറ്റ് കീകൾ അമർത്തേണ്ടതില്ല.
6. ഓരോ ചോദ്യത്തിനും നിങ്ങൾക്ക് **3 ശ്രമങ്ങൾ** ലഭിക്കും. ഓരോ തവണയും ഉത്തരം തെറ്റുമ്പോൾ ചോദ്യം ആവർത്തിച്ചു കേൾപ്പിക്കും.
7. മൂന്ന് ശ്രമങ്ങളും പരാജയപ്പെട്ടാൽ ശരിയായ ഉത്തരം വായിച്ചു കേൾപ്പിക്കും (ഗെയിം മോഡിൽ ഒഴികെ), തുടർന്ന് അടുത്ത ചോദ്യത്തിലേക്ക് നീങ്ങും.

## ⚙️ ക്രമീകരണങ്ങൾ (Settings)

പ്രധാന മെനുവിന്റെ താഴെ കാണുന്ന **Settings** ബട്ടണിൽ **Tab** ഉപയോഗിച്ച് എത്തി **Space Bar** അമർത്തുക. ക്രമീകരണങ്ങളുടെ ഉള്ളിലുള്ള ടാബ് ക്രമം ഇപ്രകാരമാണ്:
1. **പ്രയാസം (Difficulty Slider)**: ആരോ കീകൾ (Arrow Keys) ഉപയോഗിച്ച് പ്രയാസ നില മാറ്റുക.
2. **ഭാഷ മാറ്റുക (Reset Language)**: ഭാഷ തിരഞ്ഞെടുക്കുന്നതിനുള്ള വിൻഡോ വീണ്ടും തുറക്കാൻ സഹായിക്കുന്നു.
3. **സഹായം (Help)** & **വിവരങ്ങൾ (About)**.
4. **ശരി / റദ്ദാക്കുക (OK / Cancel)**: വരുത്തിയ മാറ്റങ്ങൾ സേവ് ചെയ്യാൻ **OK** അമർത്തുക, അല്ലെങ്കിൽ ഒഴിവാക്കാൻ **Cancel** അമർത്തുക.

## 🧪 ഇഷ്ടാനുസൃത ചോദ്യങ്ങൾ ചേർക്കൽ (എക്സൽ)

`[അധ്യാപകർക്കായി]`

അധ്യാപകർക്ക് കുട്ടികൾക്കായി പ്രത്യേക പാഠങ്ങൾ എക്സൽ (.xlsx) ഫയൽ വഴി അപ്‌ലോഡ് ചെയ്യാവുന്നതാണ്. ഇതിനായി പ്രധാന മെനുവിന്റെ താഴെയുള്ള **Upload** ബട്ടൺ ക്ലിക്ക് ചെയ്യുക. എക്സൽ ഫയലിൽ **question**, **operands**, **equation** എന്നീ മൂന്ന് കോളങ്ങൾ ഉണ്ടായിരിക്കണം. ഒപ്പറാന്റുകൾ നിർണ്ണയിക്കാൻ താഴെ പറയുന്ന ചിഹ്നങ്ങൾ ഉപയോഗിക്കുക:

| ചിഹ്നം (Syntax) | അർത്ഥം | ഉദാഹരണം |
| :--- | :--- | :--- |
| **`*`** | ഏതെങ്കിലും ഒരു റാൻഡം സംഖ്യ | `*` |
| **`,`** | നൽകിയിരിക്കുന്ന പട്ടികയിൽ നിന്നും ഒരു സംഖ്യ തിരഞ്ഞെടുക്കുന്നു | `2,4,6,8` |
| **`:`** | രണ്ട് സംഖ്യകൾക്കിടയിലുള്ള ഒരു റാൻഡം സംഖ്യ | `10:20` |
| **`;`** | നൽകിയിരിക്കുന്ന പരിധിയിലെ സംഖ്യകൾ ഗുണിക്കുന്നു | `2;10:20` |

ഇതിനായുള്ള മാതൃകകൾ ഔദ്യോഗിക ഗിറ്റ്‌ഹബ് പേജിൽ ലഭ്യമാണ്: [Maths-Tutor-QT-V2 Templates](https://github.com/Devikavr24/Maths-Tutor-QT-V2)

## ♿ ആക്സസിബിലിറ്റി സവിശേഷതകൾ

മാത്സ്-ട്യൂട്ടർ ഏറ്റവും ഉയർന്ന ആക്സസിബിലിറ്റി മാനദണ്ഡങ്ങളിലാണ് നിർമ്മിച്ചിരിക്കുന്നത്:
* **എല്ലായ്പ്പോഴും പ്രവർത്തനക്ഷമമായ വോയ്‌സ് (Always-On Speech)**: ടെക്സ്റ്റ്-ടു-സ്പീച്ച് പൂർണ്ണമായും സംയോജിപ്പിച്ചിരിക്കുന്നു, ഓരോ പ്രവർത്തനവും ഉടൻ വായിച്ചു കേൾപ്പിക്കുന്നു.
* **സ്ക്രീൻ റീഡർ അനുയോജ്യം**: സിസ്റ്റം സ്ക്രീൻ റീഡറുകളുമായി പൂർണ്ണമായും ഒത്തുപോകുന്നു.
* **ഓരോ അക്കവും വായിക്കൽ (Digit-by-Digit Reading)**: വലിയ സംഖ്യകൾ കുട്ടികൾക്ക് എളുപ്പത്തിൽ മനസ്സിലാക്കാൻ ഓരോ അക്കങ്ങളായി വായിച്ചു കേൾപ്പിക്കാൻ സാധിക്കുന്നു.
* **ഉയർന്ന കോൺട്രാസ്റ്റ് (High Contrast)**: ഭാഗിക കാഴ്ചയുള്ളവർക്കായി വിൻഡോസ്/ലിനക്സ് ഹൈ-കോൺട്രാസ്റ്റ് തീമുകളെ അനുകൂലിക്കുന്നു.

## 🔍 പ്രശ്നപരിഹാരം (Troubleshooting)

**പ്രശ്നം: എനിക്ക് ശബ്ദമോ ചോദ്യങ്ങളോ കേൾക്കാൻ കഴിയുന്നില്ല.**
* **കാരണം**: നിങ്ങളുടെ സ്പീച്ച് എഞ്ചിൻ ഇൻസ്റ്റാൾ ചെയ്തിട്ടില്ലായിരിക്കാം അല്ലെങ്കിൽ ഓഡിയോ മ്യൂട്ട് ആയിരിക്കാം.
* **परिഹാരം**: `espeak-ng` ഇൻസ്റ്റാൾ ചെയ്തിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക, ഒപ്പം സിസ്റ്റം വോളിയം പരിശോധിക്കുക.

**പ്രശ്നം: എന്റെ എക്സൽ ഫയൽ ലോഡ് ആകുന്നില്ല.**
* **കാരണം**: ഫയൽ .xlsx ഫോർമാറ്റിൽ അല്ലായിരിക്കാം അല്ലെങ്കിൽ കോളം പേരുകൾ തെറ്റായിരിക്കാം.
* **പരിഹാരം**: ഫയൽ .xlsx ആയി സേവ് ചെയ്യുക. കോളങ്ങളുടെ പേരുകൾ `question`, `operands`, `equation` എന്ന് തന്നെയാണെന്ന് ഉറപ്പാക്കുക.

**പ്രശ്നം: എനിക്ക് ആപ്ലിക്കേഷനിൽ നിന്ന് പുറത്തു കടക്കണം.**
* **പരിഹാരം**: **Control + Q** ഒരേ സമയം അമർത്തുക. ആപ്പ് പുറത്തു കടക്കണോ എന്ന് ചോദിക്കുമ്പോൾ **Tab** അമർത്തി **Yes** എന്നതിലേക്ക് പോയി **Space Bar** അമർത്തുക.

**പ്രശ്നം: ആപ്പ് പ്രവർത്തിക്കുന്നില്ല അല്ലെങ്കിൽ പിശകുകൾ കാണിക്കുന്നു.**
